# STOCK SYSTEM

Sistema de gestion de inventario inteligente para instituciones educativas tecnicas.

## Descripcion
STOCK SYSTEM es una plataforma web diseñada para modernizar la gestion integral de inventario, prestamos, reservas y proyectos tecnicos en escuelas tecnicas.

## Caracteristicas
- Gestion de componentes electronicos y herramientas
- Sistema de prestamos y reservas
- Proyectos colaborativos
- Control de reputacion de usuarios
- Bot IA tecnico
- Alertas automaticas
- Historial de acciones

## Tecnologias
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend:** PHP (MVC)
- **Base de Datos:** MySQL
- **Servidor:** Apache (XAMPP)
